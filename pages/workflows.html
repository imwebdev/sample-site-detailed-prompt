Production Workflows

Follow these battle-tested, step-by-step workflows to set up and execute professional live productions with vMix.

Workflow 1

Single-Camera Stream Setup

The essential workflow for getting a single camera online and streaming to your platform of choice in under 30 minutes.

1

Connect Your Camera

Plug your camera into the capture card via HDMI or SDI. Open vMix and add a new Camera input — select your capture device from the list. Verify the resolution and frame rate match your camera output.

2

Configure Audio

Add your microphone as an Audio Input (or use embedded camera audio). Open the Audio Mixer and set levels — aim for peaks around -12dB to -6dB. Enable a noise gate if you're in a noisy environment.

Pro Tip Use headphones connected to the Bus B output for monitoring without feedback. Route your microphone to Bus A (stream) and Bus B (headphones).
3

Add Overlays & Graphics

Add a lower-third title using the GT Title Editor. Assign it to Overlay 1 so you can toggle it on and off during your stream. Add your logo as an Image input on Overlay 2.

4

Set Up Streaming Destination

Go to Stream Settings and select your platform (YouTube, Twitch, Facebook, or Custom RTMP). Paste your stream key and set the bitrate — 4500 Kbps for 1080p30 is a good starting point.

Pro Tip Run a 60-second test stream first using a private/unlisted event. Check for dropped frames and audio sync before going live to your audience.
5

Go Live & Monitor

Click the Stream button to begin. Monitor the streaming status bar for dropped frames, bitrate stability, and CPU usage. Use the web controller on your phone for quick overlay toggles.

Workflow 2

Multi-Camera Live Event

A comprehensive workflow for producing multi-camera events — conferences, church services, concerts, and corporate broadcasts.

1

Plan Your Input Layout

Map out your camera positions: wide shot, presenter close-up, audience/stage, and content slides. Decide whether to use HDMI capture cards, NDI cameras, or a mix of both. Label each input clearly in vMix.

2

Configure NDI Sources

For NDI cameras, ensure all devices are on the same network subnet. Add each NDI source as an input in vMix. Set the NDI bandwidth to the highest quality your network can support.

Pro Tip Use a dedicated Gigabit switch for NDI traffic. Avoid WiFi for video sources — wired connections ensure consistent, low-latency feeds.
3

Build Your Graphics Package

Create lower thirds, title cards, countdown timers, and social media tickers in the GT Title Editor. Prepare a spreadsheet data source for dynamic name updates between speakers.

4

Set Up Multi-View & Tally

Configure the Multi-View output on a secondary monitor showing all camera inputs. Enable tally lights so camera operators can see when they are live (red) or in preview (green).

5

Configure Recording & Streaming

Set up simultaneous recording (high-quality MP4 for post-production) and streaming (optimised bitrate for your platform). Use different encoder presets for each output.

6

Run a Full Rehearsal

Execute a complete technical rehearsal before the event. Test every camera switch, overlay, audio source, and stream destination. Verify recording is capturing properly. Document any issues and fix them before show time.

Pro Tip Create a vMix preset file (.vmix) after rehearsal. This saves your entire production setup and can be reloaded instantly if anything goes wrong before the event.

Workflow 3

Replay & Highlights Production

Set up instant replay and highlight compilation for sports, esports, and any production that benefits from showing key moments again.

1

Enable Instant Replay

Open the Instant Replay panel and select which camera inputs to record for replay. Set the replay storage location to a fast SSD drive with enough capacity for the duration of your event.

2

Configure Replay Channels

Select up to 4 camera angles for replay recording. Each angle records independently, allowing you to choose the best perspective during playback. Set the recording quality to match your production resolution.

Pro Tip Use an NVMe SSD for replay storage — it handles the simultaneous read/write demands of multi-angle replay far better than traditional SATA SSDs.
3

Mark Key Moments

During the live production, press the Mark In/Out buttons (or use a hardware controller like an X-keys) to tag key moments — goals, highlights, big plays. These markers let you quickly find and replay the best moments.

4

Play Back in Slow Motion

When you want to show a replay, select the event from your marked list, choose the camera angle, set the playback speed (50%, 25%, or frame-by-frame), and cut to the replay input on your program output.

5

Build a Highlight Reel

After the event, use the marked events to compile a highlight reel. Export individual clips or string them together using vMix's playlist feature for a polished post-event package.

Pro Tip Add a "REPLAY" overlay graphic on a dedicated overlay channel that automatically appears whenever you switch to the replay input. Use a vMix shortcut to trigger both the input switch and the overlay simultaneously.
