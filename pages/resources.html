Reference

Live Production Glossary

B C D F G H I K L M N O P R S T V

B

Bitrate
The amount of data transmitted per second in a video stream, measured in Kbps or Mbps. Higher bitrate generally means better quality but requires more bandwidth.
Bus (Audio)
A routing path for audio signals. vMix offers multiple audio buses allowing different mixes for streaming, recording, and headphone monitoring.

C

Capture Card
A hardware device that converts HDMI or SDI video signals into data that software like vMix can process. Available in PCIe and USB form factors.
Chroma Key
A technique for compositing two images together by removing a solid colour (usually green or blue) from one image, revealing the background behind it.

D

DSK (Downstream Key)
A keying layer applied after the main program mix, typically used for persistent graphics like logos, bugs, or tickers.

F

Frame Rate
The number of individual frames displayed per second (fps). Common rates include 24fps (film), 30fps (broadcast), and 60fps (sports/gaming).
Full Duplex
The ability to send and receive data simultaneously. Important for network-based video protocols like NDI and SRT.

G

Genlock
A method of synchronizing the timing of multiple video sources so they can be mixed without glitches. vMix handles this internally via software processing.
GT Title Editor
vMix's built-in graphic title editor for creating lower thirds, tickers, scoreboards, and animated text overlays with data source connectivity.

H

HDMI
High-Definition Multimedia Interface — a standard for transmitting uncompressed video and audio from a source to a display or capture device.

I

Input
Any source added to vMix — cameras, videos, images, audio, web pages, NDI sources, etc. Each input appears as a thumbnail in the input panel.
Instant Replay
A feature that continuously records selected camera inputs and allows playback of recent footage at any speed, commonly used in sports production.

K

Key / Keying
The process of compositing video by removing a specific colour or luminance range, making parts of the image transparent to reveal layers beneath.

L

Lower Third
A graphic overlay occupying the lower portion of the screen, typically displaying a person's name, title, or other contextual information.

M

Multi-View
A display mode showing multiple input sources simultaneously on a single monitor, allowing the operator to monitor all cameras at once.

N

NDI (Network Device Interface)
A protocol developed by Vizrt (formerly NewTek) for sending high-quality, low-latency video over standard IP networks. Widely supported by vMix.

O

Overlay
A graphics layer placed on top of the program output. vMix supports four independent overlay channels for logos, tickers, lower thirds, and PiP windows.

P

PiP (Picture-in-Picture)
A smaller video window placed over the main program output, often used to show a secondary camera angle or remote caller.
Program Output
The final mixed video signal that is sent to stream, recording, or external outputs. What the audience sees.
Preview
A buffer showing the next input that will go live when a transition is triggered. Part of the Program/Preview (A/B) switching model.

R

RTMP (Real-Time Messaging Protocol)
A protocol used for streaming live video to platforms like YouTube, Facebook, and Twitch. The most widely supported streaming protocol.
RIST
Reliable Internet Stream Transport — a protocol designed for reliable video transport over unpredictable networks, similar to SRT.

S

SDI (Serial Digital Interface)
A professional broadcast standard for transmitting uncompressed video over coaxial cables. Common in broadcast environments using BNC connectors.
SRT (Secure Reliable Transport)
An open-source protocol for low-latency video delivery over unpredictable networks. Supports encryption and error correction for contribution feeds.
Stinger Transition
An animated video clip used as a transition between two sources. The animation plays over the cut point, creating a polished branded transition effect.

T

Tally Light
An indicator on a camera showing whether it is live (red) or in preview (green). vMix supports tally via NDI, HTTP API, and hardware controllers.
Transition
The visual effect used when switching between two video inputs — common types include cut, dissolve, wipe, slide, and stinger.

V

Virtual Set
A 3D computer-generated environment where a chroma-keyed presenter is composited into a virtual studio. vMix includes a built-in virtual set engine.
vMix Call
A built-in video calling feature that allows remote guests to join a production through a web browser with up to 8 simultaneous callers in HD.
